Database Infrastructure

Counties were first grouped by county/county labels; vaccination rates were averaged and the population/ vaccination counts were summed. Data was then imported into Google Colab where columns were narrowed down to county/county, total population and vaccination rate.

Machine Learning

We chose to use a neural network to predict vaccination rates because the model behaves like a regression model, where a dependent variable or target (i.e., vaccination rates) can be predicted from independent variables or features (i.e., demographic information).
