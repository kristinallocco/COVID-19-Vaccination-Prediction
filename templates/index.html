COVID-19 Vaccination Rate Analysis

The purpose of this analysis is to gain insight into what factors impact COVID-19 vaccination rates for New York State residents.

This project was made possible through the use of vaccination data as well as New York State census data to analyze COVID-19 vaccination rates across several demographics among New York State Residents, including, income, education, and ethnicity. 

To determine the variables with the most substancial effects, a neural network model to predict the COVID-19 vaccination rates of New York State residents. The model emulates a regression model, where the dependant variable (vaccination rates) can be derived from independent variables (demographic information).
